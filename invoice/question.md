### 売上レポート生成器

あなたは、ある小売店の売上レポートを生成するシステムを開発することになりました。このシステムは、日々の売上データを受け取り、それを異なる形式（TEXT、HTML、JSON）で出力する必要があります。将来的には他の形式にも対応できるよう、拡張性を持たせることが求められています。

### 要件

1. `SalesReportGenerator` クラスを作成してください。このクラスは以下のメソッドを持つ必要があります：
    - `initialize(sales_data)`: 売上データを受け取ります。
    - `generate_report(format)`: 指定された形式でレポートを生成し、結果を返します。
2. 売上データは以下の形式の配列で与えられます：
    
    ```ruby
    
    [
      { date: "2023-03-01", amount: 1000 },
      { date: "2023-03-02", amount: 1500 },
      { date: "2023-03-03", amount: 1200 },
    ]
    
    ```
    
3. 以下の形式でレポートを生成できるようにしてください：
    - TEXT: プレーンテキスト形式
    - HTML: シンプルなHTML形式
    - JSON: JSON形式
4. レポートには以下の情報を含める必要があります：
    - 総売上額
    - 日別の売上額
    - データ期間（最初の日付と最後の日付）
5. 新しい出力形式を簡単に追加できるよう、拡張性を考慮した設計にしてください。

### 例

```ruby
ruby
Copy
sales_data = [
  { date: "2023-03-01", amount: 1000 },
  { date: "2023-03-02", amount: 1500 },
  { date: "2023-03-03", amount: 1200 }
]

generator = SalesReportGenerator.new(sales_data)

puts generator.generate_report("TEXT")
puts generator.generate_report("HTML")
puts generator.generate_report("JSON")

```
